# DECISIONS.md（確定済みの方針）

## [2026-04-03] エージェント構成
- QAエージェントはOSCAR2側（管理レイヤー）に属する
- ストラテジーエージェントは作業レイヤーに属する
- 両エージェントともOpus 4.6を使用

## [2026-04-03] 品質管理方針
- QAエージェントは作業者から独立した立場で品質判定を行う
- 品質基準の緩和にはKojiの明示的承認が必要
- ローカルのみの検証で合格判定を出さない

## [2026-04-03] タスク管理方針
- 参謀チャットで方針を決定し、自走可能な指示書を作成してからキューに投入
- タスクが止まったら「参謀待ち」に移して次タスクへ
- リトライ上限は3回。超過したらエスカレーション

## [2026-04-04] SNS方針
- 無関係なSNS投稿を埋め込むくらいならembed=0+画像フォールバックの方がマシ
- テーマ（結婚/離婚等）に直接言及していない投稿はadoptしない
